# Wreckfest
### Install notes
On first start Wine will print some error messages, ignore them

### Specifications
|      | Recommended | Minimal |
|------|-------------|---------|
| RAM  | 1024 MB     | 768 MB  |
| Disk | 10000 MB    | 6000 MB |

### Server Ports
| Port    | default |
|---------|---------|
| Game    | 33540   |
| Steam   | 27015   |
| Query   | 27016   |

#### Mods/Plugins may require ports to be added to the server.
